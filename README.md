# Prompt Type 1 – Comparative Analysis Prompt

## Definition

Comparative prompts compare two or more concepts by identifying similarities and differences.

## Prompt Examples

> “Compare traditional manufacturing and smart manufacturing systems.”

> “Compare machine learning and deep learning in industrial automation.”

## Generated Output

| Feature | Traditional Manufacturing | Smart Manufacturing |
|---|---|---|
| Monitoring | Manual | Automated |
| Maintenance | Reactive | Predictive |
| Efficiency | Moderate | High |
| Energy Usage | Higher | Optimized |

---

# Prompt Type 2 – Experiential Perspective Prompt

## Definition

This prompt asks the AI to respond from a specific perspective or experience.

## Prompt Examples

> “As a factory manager, explain the benefits of predictive maintenance.”

> “From an engineer’s perspective, describe the importance of IoT in manufacturing.”

## Generated Output

Predictive maintenance helps factory managers reduce unexpected machine failures, improve productivity, and lower maintenance costs by detecting issues early using AI and sensor data.

---

# Prompt Type 3 – Everyday Functioning Prompts

## Definition

These prompts explain how technology works in everyday applications.

## Prompt Examples

> “Describe how AI is used in everyday factory automation.”

> “How are sensors used in smart manufacturing systems?”

## Generated Output

Sensors collect real-time machine data such as temperature, pressure, and vibration. AI analyzes this data to identify faults and improve production efficiency.

---

# Prompt Type 4 – Universal Prompt Structures

## Definition

Universal prompts follow standard reusable structures for explanation and learning.

## Prompt Examples

> “Explain predictive maintenance in simple terms with an example.”

> “Explain the process of smart manufacturing step by step.”

## Generated Output

Predictive maintenance uses AI and sensors to monitor machines and predict failures before they occur. For example, vibration sensors in motors can detect unusual patterns and alert technicians before breakdowns happen.

---

# Prompt Type 5 – Prompt Refinement and Prompt Size Limitations

## Definition

Large prompts are divided into smaller manageable tasks for better AI performance.

## Prompt Examples

### Large Prompt
> “Explain 50 applications of AI in manufacturing.”

### Refined Prompt
> “Explain 10 applications of AI in manufacturing at a time.”

---

### Large Prompt
> “Generate a complete industrial automation report.”

### Refined Prompt
> “Generate the introduction section of the industrial automation report.”

## Generated Output

Breaking large prompts into smaller sections improves response clarity, reduces overload, and generates more accurate outputs.
